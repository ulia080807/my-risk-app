# Пользовательские сценарии

## Основной сценарий: Оценка риска

**Как** пользователь 50 лет с гипертонией,
**Я хочу** оценить свой риск инсульта,
**Чтобы** понимать, на что обращать внимание.

**Шаги:**
1. Открываю приложение
2. Заполняю анкету:
   - Возраст: 50
   - Давление: 150/95
   - Вес: 90 кг
   - Рост: 175 см
   - Курение: нет
   - Диабет: нет
3. Получаю результат: "Умеренный риск (12%)"
4. Вижу рекомендации:
   - Контролировать давление
   - Снизить вес на 5 кг
   - Проверить холестерин

## Сценарий: Мониторинг состояния

**Как** пациент после микроинсульта,
**Я хочу** отслеживать свои показатели,
**Чтобы** предотвратить повторный инсульт.

**Шаги:**
1. Ежедневно вношу данные давления
2. Раз в неделю взвешиваюсь
3. Приложение строит графики
4. При отклонениях получаю уведомление

## Сценарий для врача

**Как** врач-невролог,
**Я хочу** видеть динамику пациентов,
**Чтобы** корректировать лечение.

**Шаги:**
1. Вхожу в веб-панель
2. Вижу список пациентов
3. Выбираю пациента
4. Смотрю графики показателей
5. Отправляю рекомендации через приложение
